So, yesterday I installed Claude Code for VS Code and I feel I have opened a pandoras box. It's great and I am honestly amazed of what it can do. 

I started out with small changes to my website but soon recognized that it's probably way more powerful. So I thought, lets test this vibecoding thing!

New folder -> no files, just me and Claude Code for VS Code. I set it the task to provide me the code of a webpage with a chatwindow. It should communicate via the API with Anthropic Claude. And after a few trials it worked! Below are two screenshots of the final program, running with node.js and jekyll. I didn't touch any of the code, just chattet with the bot.

First I was amazed by this. So anyone without any coding background can really build stuff fast! Endless possibilities.
But then, what does this mean for me and my coding journey? Will it all be over with the software industry soon? I was a bit down by this thought.
Plus, it was only fun, while claude worked the code. Once my page was up and running, I wasn't proud at all, of what I did. It felt like installing a program. And worse, I had absolutely no idea, how it worked. It just did, how boring and a bit dissapointing! For me coding is about problem solving and providing a solution, that you have to improve over and over again. But with this tool, what can I really improve or learn?

After sleeping a night and thinking a bit more about it, I am happy that I learned how to vibecode from scratch. It's also a skill and is perfect for prototyping or providing the foundation or surroundings to bring your specific app to live. But until then? I have to learn a lot of the fundamentals, before I can use such a black box of a tool.
